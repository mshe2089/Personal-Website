SAT Solver


This simple tool will brute force the entire truth table of your SAT formula.
Made to test out asynchronus page updates using ajax.

Recognized operators include conjunctions, negations, disjunctions, implications, double implications.
Recognized symbols are ['∧','¬','∨','→','↔','(',')']. Whitespaces will be ignored.
Variables can be any non-operator, non-whitespace character.

Make sure your input is syntactically correct and fully (but not redundantly) parenthesised.
(Parser is dumb and requires order of operations to be explicitly specified.)
eg. x ∨ y ∨ z and (x) ∨ (y ∨ z) are not okay, but (x ∨ y) ∨ z is.
